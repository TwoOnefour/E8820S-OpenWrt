# Description
这是一个ZTE-E8820S自用固件仓库，源码采用[LEAN](https://github.com/coolsnowwolf/lede.git), 官方源码编译为 `linux 6.6 `内核

配置为
- zerotier
- usb
- 开源和闭源wifi
- passwall (shadowsocks-rust,ssr,ss,xray)
- smartdns
- kmod-cryptodev (Cryptographic Hardware Accelerators)
- kmod-crypto-aead
- kmod-crypto-gcm
- openssh-sftp-server
