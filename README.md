# Description
这是一个ZTE-E8820S自用固件仓库，官方源码采用23.05版本

配置为
- zerotier
- usb
- 开源和闭源wifi
- passwall (shadowsocks-rust,ssr,ss,xray)
- smartdns
- kmod-cryptodev (Cryptographic Hardware Accelerators)
- kmod-crypto-aead
- kmod-crypto-gcm
- openssh-sftp-server
